# Global CO2 Emissions KPI Dashboard (2015–2023)

Clean OWID dataset (~90 filtered rows) for climate-tech Net-Zero analysis.

## Key Insights
- China's 2023 CO2: 11.9B kt (32.5% global share), +4.2% YoY growth.
- India's emissions up 33% since 2015—focus for industrial decarbonization.

## Tools
- MS Excel (Power Query ETL, PivotTables, Slicers for aggregation/synthesis).
- SQL (ETL/growth queries in /sql_queries).
